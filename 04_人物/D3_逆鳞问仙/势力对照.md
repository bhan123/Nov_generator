# 逆鳞问仙 — 势力对照表

> 本书势力架构基于 SHI-002（天骄阁）+ SHI-003（凡尘派）双势力对峙设计。

---

## 一、势力总览

### 1.1 天骄阁（SHI-002 — 终极反派组织）

**意识形态**：修真文明始于贵族血脉，凡人不配修炼。

**实际真相**：两千年前，天骄阁窃取凡人修真成果（凡尘诀），抹去起源历史，把修真变成"贵族专属"。

| 层级 | 名称 | 关键人物 | 与主角关系 |
|------|------|----------|-----------|
| 总阁 | 天骄阁总坛 | 阁主·天骄真人 | 卷七终极 Boss |
| 分阁 | 东域分阁 / 南域分阁 / 西域分阁 / 北域分阁 / 中域分阁 | 五位分阁主 | 各地打压凡尘派余脉 |
| 下辖组织 | 摘星楼（底层监控） | 摘星楼楼主 | 卷一陷害主角 |
| 下辖组织 | 天罡宗（打手） | 宗主 | 卷二追杀主角 |
| 下辖组织 | 幽冥殿（暗杀） | 殿主 | 卷三-卷四暗杀主角 |
| 下辖组织 | 丹塔（丹药垄断） | 塔主 | 卷五-卷六垄断丹药 |
| 下辖组织 | 炼器阁（装备垄断） | 阁主 | 卷六垄断装备 |

**叙事功能**：天骄阁是"血统垄断+神权叙事"的具象化反派，与凡尘纪原作中天骄阁的设定一致——但 D3 把天骄阁的"伪善程度"提升到极致：他们表面是"修真文明的秩序守护者"，实际是"窃取凡人修真成果的小偷集团"。

### 1.2 凡尘派（SHI-003 — 被剿灭的真相派）

**理念**：修真起源于凡人，凡人均可修炼。

**现状**：两千年前被天骄阁剿灭，零星传承。

| 层级 | 名称 | 关键人物 | 与主角关系 |
|------|------|----------|-----------|
| 派主级 | 凡尘派派主 | （两千年前的凡尘派主，已故） | 主角的师祖 |
| 元老级 | 凡尘派元老 | 玄一老人 / 玄二老人 | 卷六发现真相的关键知情人 |
| 传承级 | 主角父亲·裴去病的父亲（已故） | 裴孤山 | 卷六灵魂现身 |
| 传承级 | 主角本人 | 裴去病 | 凡尘派末裔 |
| 旁支 | 老矿监赵叔 | 赵潜 | 卷一-卷四暗中相助 |
| 旁支 | 凡尘派故旧若干 | 各分布于七层地图 | 卷六-卷七集结 |

**叙事功能**：凡尘派在本书中不再是"被剿灭后无人问津的传说"，而是"暗中守护真相的隐秘组织"。裴去病的父亲裴孤山（卷六灵魂现身）是凡尘派的当代传人——这是 SH-013 父爱如山反转的核心。

## 三、势力间的核心冲突线

### 3.1 凡尘派 vs 天骄阁（文明级冲突）

**两千年前**：凡尘派创始"凡尘诀"，让凡人皆可修真。天骄阁窃取凡尘诀，抹去起源历史，把修真变成贵族专属。凡尘派被剿灭。

**本书开始**：凡尘派残存的末裔裴去病（主角）被冤枉除名，被迫流亡。他在流亡中逐步发现自己是凡尘派遗脉——这是卷六的真相大白。

**本书高潮（卷七）**：裴去病当众公开"修真起源于凡人"的真相，天骄阁的"血统论"叙事当场崩塌。

### 3.2 天骄阁内部分裂

天骄阁不是铁板一块——卷四-卷六期间，分阁主之间出现裂痕：
- 东域分阁主：被主角的"以德报怨"（SH-018）感动，卷五暗中倒戈
- 中域分阁主：被主角的"父债子还"（SH-017）震撼，卷六公开质疑总阁
- 西域分阁主：天骄阁的死忠，最终决战时被击杀

### 3.3 凡尘派内部的觉醒

凡尘派的元老们起初对主角不信任（他父亲裴孤山当年被怀疑是叛徒）。卷六灵魂现身 + 裴去病的觉醒，让凡尘派残部终于相信"凡尘派的希望还在"。

## 四、势力对照 — 与凡尘纪的差异

| 维度 | 逆鳞问仙(D3) | 凡尘纪 |
|------|-------------|--------|
| 反派势力 | 天骄阁（凡尘纪同款，但伪善程度提升） | 天骄阁 |
| 真相派 | 凡尘派（被剿灭的真相派） | 凡尘派 |
| 主角出身 | 凡尘派末裔（被冤枉除名） | 矿奴之子（底层废物） |
| 与反派关系 | 主角父亲被反派诬陷为主动出卖凡尘派 | 主角父亲失踪留下残简 |
| 主角成长路径 | 被冤枉→自证→复仇→公开真相 | 废灵根→觉醒→立志→公开真相 |
| 调性 | 暗黑+悲壮+反英雄 | 沉+燃+思想 |

**关键差异点**：D3 的天骄阁比凡尘纪原作中的天骄阁更"伪善"——他们不是单纯的反派，而是"窃取凡人修真成果却自诩为秩序守护者"的精致利己者。这是 D3 调性更暗黑的核心原因。

---

## 五、势力对照表的特殊说明

D3 的势力对照表与 D1 不同：

- **D1 的势力是"古族派系内斗"**——多极、纵横捭阖，爽点在政治手腕
- **D2 的势力是"学院代行"**——单极、内向推进，爽点在伙伴羁绊
- **D3 的势力是"双极对峙"**——天骄阁 vs 凡尘派，正邪分明，爽点在文明级真相大白

读者在翻开 D3 的势力表时，应当理解为：
- **天骄阁**：精致的伪善反派——他们不是"坏人"，而是"窃取真相的人"
- **凡尘派**：悲壮的真相守护者——他们不是"圣人"，而是"坚持'凡人都能修炼'的人"

这两种势力设定，让 D3 在爽感上既有"凡尘纪原作"的思想深度，又有"反英雄"叙事的暗黑调性。
