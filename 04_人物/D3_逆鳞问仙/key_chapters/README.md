- 第22章《你信仰的反转》：裴去病在摘星楼公开证据，逼分堂执事面对天骄阁窃取凡尘诀的历史；主爽点为反派真相反转与信仰翻转；钩子是幽冥殿三名死士追来。

- 第55章《凡尘派遗迹》：裴去病踏遍第五处故地，在父亲手稿与元老见证下开启凡尘诀完整版；主爽点为父爱如山最终反转；钩子是天骄阁总坛的公开对质。
- 第77章《扫坐现场》：天骄总坛对质，裴去病扫落象征血统秩序的阁主宝座，当众公开修真起源；主爽点为扫坐现场+文明抉择+真相雪冤；钩子是开放功法后各地新秩序的余波。
