## 1. 主线矛盾

### 1.1 表层矛盾（卷一-卷四显线）

**叛门除名 + 冤屈自证**

裴去病本是青云宗内门弟子，修炼天赋属上等，被宗门长老寄予厚望。但在他二十岁那年，一封匿名信指控他"勾结魔门，意图颠覆宗门"。宗门召开公审大会，裴去病百口莫辩——关键证人（他的师兄裴孤云）当众指证他"在藏经阁偷窃禁术"。

裴去病被当众废除修为，逐出师门。他的父亲裴孤山（凡尘派末裔·真实身份从未公开）听到这个消息后，吐血而亡。

**关键时间锁**：裴去病立下誓言——三年内，他会回来，让诬陷他的人"社会性死亡"。

### 1.2 深层矛盾（卷四-卷七暗线）

**凡尘派遗脉 + 修真起源真相**

裴去病在流亡中逐步发现：
1. 他的父亲裴孤山不是普通青云宗弟子，而是凡尘派的当代传人
2. 父亲的死不是气死的——是被天骄阁的暗杀组织幽冥殿所害
3. 师兄裴孤云的指证是被天骄阁收买的结果
4. 天骄阁窃取了凡尘诀，把修真变成贵族专属

**深层目标**：揭露天骄阁窃取凡尘诀的真相 → 让主角继承父亲遗志 → 在第七卷公开"修真起源于凡人"的真相 → 让天骄阁的"血统论"叙事当场崩塌。

## 4. 卷纲概述（锁定 7 卷 + 每卷主题 + 核心爽点）

### 卷一·青云废徒·冤屈开局（≈40 章 / 40 万字）

**核心爽点**：SH-017 父债子还（伏笔）+ SH-003 隐藏身份揭露（前奏）

**开篇钩子（第 1-3 章）**：
- 第 1 章：青云宗公审大会，裴去病被师兄裴孤云当众指证"勾结魔门"。关键证人出场，但证人陈述有漏洞——裴去病敏锐地抓住了漏洞，但被宗门长老以"狡辩"驳回。
- 第 2 章：裴去病被废除修为，逐出师门。父亲裴孤山（凡尘派末裔）在山中听到消息，吐血。当夜，幽冥殿的暗杀者潜入裴家，裴孤山重伤身亡——临死前把一枚玉佩交给裴去病。
- 第 3 章：裴去病在父亲坟前立下誓言。玉佩在月光下闪出微光——这是凡尘派传承的信号。

**中段（第 4-30 章）**：
- 裴去病流亡到凡人界底层
- 在一个凡尘派的小据点（老矿监赵叔处）得到父亲的遗物——一本没有封面的《凡尘诀》残简
- 父亲留下的遗书揭示："儿啊，你不是普通青云弟子——你是凡尘派的末裔"
- 裴去病开始暗中修炼凡尘诀——但他的经脉已经被废除，必须从头开始

**卷末高潮（第 31-40 章）**：
- 青云宗派出追兵，要彻底灭口裴去病
- 裴去病被逼到绝境，凡尘诀残简突然激活——他身上爆发出一股凡尘派末裔才有的血脉气息
- 以"凡人"之力 + 残简初开之力，裴去病反杀追兵
- **关键伏笔**：追兵头目临死前说："你的血……是天骄阁要的血"

**主调**：虐 + 燃。开局即被冤枉 + 废除修为，是反英雄叙事的标准开局。

## 7. 核心金手指机制详解

### 7.1 KZ-001 信息差（凡尘真源）

**触发**：裴去病知道天骄阁不知道的修真起源真相

**关键信息**：
- 修真起源于凡人（凡尘诀祖师留下的遗训）
- 天骄阁窃取了凡尘诀
- 天骄真人是两千年前的叛徒
- 凡尘诀是所有凡人都能修炼的功法

**爽度规则**：信息差越大，主角对天骄阁的认知压制越强；最高级是让对方整个叙事体系崩塌

**限制**：需要场景展现，不能纯说；每次释放要配合至少一个震惊的见证者

**use_frequency**：每卷至少爆发 3 次

### 7.2 KZ-008 血脉觉醒（凡尘派遗脉）

**触发**：裴去病的父母之一是凡尘派末裔，血脉沉睡 → 特定剧情触发觉醒

**三次觉醒时间表**：
- 卷三末：第一次半觉醒（血气爆发，反杀幽冥殿暗杀者）
- 卷五末：第二次完全觉醒（血气+战力质变，以德报怨分阁主）
- 卷七中：第三次终极觉醒（凡尘派末裔的全部血脉 + 凡尘诀完整版 = 神祇级）

**爽度规则**：血脉浓度越高 → 觉醒后战力加成越大 → 反派认知崩塌越深

**限制**：
- 必须配合身世揭秘剧情线（卷六父亲灵魂现身）
- 觉醒不可提前（卷三半觉醒、卷五完全觉醒、卷七终极觉醒）
- 觉醒后战力膨胀要克制

### 7.3 金手指组合 vs 凡尘纪的差异

凡尘纪走 KZ-001 信息差（反血统论发动机）+ KZ-005 有限回档（剧情反转器）。
D3 逆鳞问仙走 KZ-001 信息差（保留！这是凡尘纪灵魂）+ KZ-008 血脉觉醒（替换 KZ-005）。
**关键差异**：D3 主动移除了 KZ-005 回档机制——反英雄叙事不需要"剧情回档"这种"再来一次"的爽感，而需要"不可逆的命运 + 灵魂层面的觉悟"的更深层爽感。

## 11. 写作纪律（500 字中不能出现什么）

### 11.1 禁用元素（500 字内不得出现）

- **不可出现凡尘纪原作主角名** —— 李遗、矿奴、李凡尘等凡尘纪专属名词一概不出现
- **不可出现"双生武魂""昊天锤""蓝银皇"等斗罗专属词汇** —— 本书用"凡尘派"修真体系
- **不可出现"佛怒火莲""异火榜""斗气"等斗破专属词汇** —— 本书用"修真/灵气"体系
- **不可使用"系统""叮""宿主"等系统流词汇** —— 本书是纯古典反英雄玄幻
- **不可使用现代网络用语** —— "皮一下""真香""大佬"等禁止
- **不可写主角"爽文式"开挂** —— 本书是反英雄叙事，主角不能一帆风顺

### 11.2 必须遵循的叙事纪律

- 每 500 字内至少 1 个动作描写（拳头紧握 / 眼眸微冷 / 灵气一振）
- 每 1000 字内至少 1 个环境描写（矿底阴冷 / 凡城黑水 / 天骄阁总坛）
- 每章末必须留下悬念钩子
- 反派出场必须有"伪善面具下露出獠牙"的视觉描写
- 主角反击必须有"被冤枉很久后第一次爆发"的情绪铺垫
- 父亲（裴孤山）出场必须有"沉默+承受+为儿"的复合形象
- 反派天骄阁出场必须有"道貌岸然+私下算计"的对比描写

### 11.3 章节结构纪律

- 每章 3000 字 ± 200 字
- 每 5 章一个小高潮，每 15 章一个中高潮，每 30 章一个大高潮
- 每卷首章必须包含：开篇钩子 + 卷主题 + 卷核心爽点预告
- 每卷末章必须包含：卷级 Boss 战 + 主角身份升级 + 下一卷钩子

### 11.4 角色语言纪律

- 主角裴去病：阴沉+内敛+关键场景爆金句
- 父亲裴孤山：沉默+一句顶一万句（卷六灵魂现身时大量对白）
- 玄一老人：严谨+对凡尘派的敬重+对裴孤山的歉疚
- 裴孤云（师兄）：前期诬陷时理直气壮→卷五被感化后羞愧难当
- 东域分阁主：被以德报怨时眼神复杂→卷六暗中倒戈时坚定
- 天骄真人：伪善+哲学化语气（"血统论才是修真文明稳定的根基"）

### 11.5 禁忌语句

- "他心中一惊"（不能纯心理）
- "一股强大的灵气扑面而来"（必须有具体感官）
- "众人倒吸一口凉气"（不能套路化震惊）
- "此人实力深不可测"（必须有具体对比）
