## 章节清单与功能
- 第1章《穿越开局》：武魂觉醒仪式上沈夜星被判为蓝银草废武魂；主爽点为废武魂反转的低开；钩子是无封面小册子夜间发光。
- 第5章《史莱克入学考》：沈夜星、柳蘅、牛铁柱以底层组合参加学院入学考；主爽点为学院派知识碾压与伙伴初成；钩子是凡尘心副本开启。
- 第10章《双生武魂首曝光》：实战考核中昊天锤在绝境觉醒，沈夜星击退蓝电霸王龙天才；主爽点为废武魂反转+双生曝光；钩子是学院调查与第2考苦难心。
- 第20章《武魂融合技出》：第4考决赛，沈夜星与柳蘅在禁赛危机后完成蓝银·昊天融合；主爽点为越级反杀与师承重建；钩子是武魂殿发布公开调查。
- 第35章《学院冠军赛》：五人组在学院冠军赛对抗武魂殿代表队，公开卷宗的一页成为胜负关键；主爽点为团队冠军爽点；钩子是大陆魂师大赛与第6考担当心。
- 第55章《魂师大赛起》：七人组参加魂师大赛，凌雁险遭灵魂收割，沈夜星首次付出回档代价；主爽点为以命换命伏笔与回档信息差；钩子是总决赛白袍祭司登场。
- 第66章《总决赛冠军》：总决赛沈夜星以双生武魂融合技击败白袍祭司，七人组开启第9考；主爽点为众望所归+双魂合璧；钩子是神祇战场和父亲灵魂。
