## 章节清单与功能
- 第1章《开局退婚》：萧媚撕毁婚书，陆家祠堂前立下三年之约；主爽点为退婚打脸；钩子是祖坟古戒即将被发现。
- 第3章《戒指觉醒》：祖坟古戒滴血，药老残魂苏醒并传授焚炎诀；主爽点为戒指老师与功法进化；钩子是斗技堂考核与陆云挑衅。
- 第15章《三方争火》：摘星坊市地心火线索引发陆劫尘、萧家分支与摘星楼三方争夺；主爽点为智斗与资源争夺；钩子是熔岩洞封闭时间提前。
- 第25章《越阶反杀》：陆劫尘以斗之气三段对决陆云手下五段高手；主爽点为越阶反杀；钩子是三月考核提前，萧媚派出天才侍卫。
- 第40章《古族初用》：七星道统大比中，陆劫尘首次以古族灵火对抗宋星河；主爽点为双火融合的半成品；钩子是决赛对阵斗师巅峰宋星河。
- 第60章《中阶迭变》：天罡宗诬陷案中，陆劫尘从被逐边缘借真证据完成反转；主爽点为认知反差与剧情反转；钩子是五域战邀请与魂族长老现身。
- 第100章《封神现场》：九重天九火归一，陆劫尘击败远古斗帝血脉并公开灵火起源；主爽点为封神现场与师徒余韵；钩子是凡尘派余脉可能连接另一部作品。
