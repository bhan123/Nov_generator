## 序言：为什么要写这份文档

网文创作已经走过二十多年。期间，斗罗、斗破、凡人、遮天、雪中、诡秘、灵境、大奉、惊悚乐园等千万级作品沉淀出一套又一套"可复用套路"。这些套路不是抄袭，而是行业累积下来的"用户情绪共识"——读者已经被训练成看见某些模式就分泌多巴胺，看见某些反模式就弃书。

但市面上的方法论有两个极端：

- **一是只讲抽象原则**："要立人设、要造冲突、要给钩子、要把握节奏"——这种话听完仍然不知道怎么动笔。
- **二是只罗列套路**："退婚打脸、戒指老爷爷、异火吞噬、武魂融合"——这种话听完不知道什么时候用、用完如何升级。

本方法论的定位是 **第三条路**：

1. **每条规律都配 3-5 个具体例句**（来自斗罗、斗破、凡尘纪、知乎高赞爽点、起点番茄爆款）。
2. **每个套路都给出"用 / 不用 / 错用"三种场景**。
3. **所有"可调用的零件"都收纳成卡片**（主角人设卡、金手指卡、爽点卡、资源卡、势力卡、伙伴卡、副本卡），可直接拼装。
4. **每张卡片都标注"等级"（T0/T1/T2）和"使用频率"**，防止主角"金手指烂尾化"。

**核心命题**：网文不是文学。网文是 **"情绪工业化生产"**。本方法论是这条流水线的操作手册。

> 一句话：**"主角人设 + 金手指 + 资源榜 + 爽点节奏 + 势力地图 + 伙伴羁绊 + 副本推进" 七大零件可拼装出任何一本 200 万字长篇玄幻爽文。**

> **结论 1**：写网文前先选品类，选品类前先看三件事——读者画像（老白/小白/女频/男频）、爽点调性（热血/轻松/烧脑/甜宠）、目标平台（起点/番茄/七猫/刺猬猫）。
>
> **结论 2**：本方法论重点服务 **A 类（东方玄幻）** 与 **C 类（仙侠）**——因为本项目《凡尘纪》属于 A+C 复合，且本类对结构化要求最高。
>
> **结论 3**：不同品类的"主角人设 + 金手指"配对不同。例如：废柴逆袭（A1）必配"戒指老爷爷+可进化功法+资源榜"；上交国家（B3）必配"国家是后盾+技术是金手指+制度是阻力"。

## 第二部分：网文的"骨内架构"——主角人设 × 金手指 × 十架构 × 十资源

### 2.1 主角人设型谱（4 大原型 + 12 配比变体）

#### 2.1.1 四大基本原型

```
原型 A：底层废物 + 觉醒者（最经典）
   - 例：萧炎（斗气消失→药老觉醒）、李隙（漏灵废根→凡尘诀激活）
   - 爽点来源：压抑-反转-打脸
   - 调性：热血、燃、励志
   - 适用：A1/A2/A4/C3

原型 B：穿越者 + 前世知识（最易上手）
   - 例：唐三（唐门外门→唐门暗器）、方平（重生前瞻+财富值系统）
   - 爽点来源：开图挂+认知碾压
   - 调性：爽、轻松、信息差
   - 适用：A2/B2/E1

原型 C：天才陨落 + 重头再来（最悲壮）
   - 例：林动（天才→废物→崛起）、许七安（现代警察→古代捕快）
   - 爽点来源：英雄落难+再创辉煌
   - 调性：热血+厚重
   - 适用：A1/C1/F1

原型 D：内部叛徒 / 弃徒（最有冲突）
   - 例：韩靖（被冤枉→复仇+真相）、李隙之父李遗（被通缉→守护传承）
   - 爽点来源：被冤枉+自证+复仇
   - 调性：压抑+反转
   - 适用：A1/A4/C1
```

```yaml

soul_keywords: [凡人皆可修炼, 修真真相, 信息差碾压, 小人物大选择]

#### 2.1.4 主角人设 6 大禁忌

1. **不能太完美**：圣人型主角让读者代入不了。必须至少有 1-2 个"小毛病"（路痴、嘴毒、贪财、好色但不下流）。
2. **不能完全没有主张**：无脑型主角（纯靠金手指）会让读者"恨铁不成钢"，必须关键时刻有"自己的判断"。
3. **不能只靠嘴遁**：30 万字以上主角必须有"动手能力"或"动手后盾"（资源/伙伴/阵法），纯嘴炮扛不到 200 章。
4. **不能人设漂移**：开篇是什么样的人，结尾不能变样。李隙从第一卷到第七卷"保护矿工"的底色不能变。
5. **不能太孤独**：必须有 3-5 个深度伙伴（不一定是恋人），完全孤独的主角撑不起群像戏。
6. **不能反智但要反套路**：写主角"知道答案"可以，但不能让读者觉得"你写得太巧了"——必须每次"巧"都有伏笔。

### 2.2 金手指型谱（T0/T1/T2 三级 + 8 大机制）

#### 2.2.1 金手指等级定义

```
T0（灵魂级）：全书 1 个，贯穿始终，决定调性
   - 例：信息差（凡尘纪 KZ-001）、戒指老爷爷（斗破 KZ-002）、双生武魂（斗罗 KZ-003）
   - 作用：让主角的"存在"有别于其他角色

T1（重要级）：全书 2-3 个，配合 T0 推进
   - 例：可进化功法（斗破 KZ-004）、有限回档（凡尘纪 KZ-005）、越级吸收（KZ-006）
   - 作用：让主角"成长"有具体路径

T2（辅助级）：全书 3-5 个，仅在特定卷使用
   - 例：前世记忆（斗罗 KZ-007）、血脉觉醒（KZ-008）
   - 作用：让主角在关键卷有"突破手段"
```

#### 2.2.2 8 大金手指机制详解

**机制 1：信息碾压型（T0）**
- **核心**：主角知道别人不知道的真相/历史/规则
- **爽点构建**：用对方规则打败对方
- **代表作**：
  - 凡尘纪 KZ-001：修真起源真相
  - 雪中悍刀行 徐凤年：北凉王继承人身份
  - 《修真四万年》李耀：联邦元婴的真相
- **写法要诀**：信息差必须"具体"——不能只说"他知道未来"，要写他"知道某个具体细节"（比如凡尘纪里"他知道照骨碑的阈值是人造的"）。
- **必配限制**：信息差要"分批释放"，每卷 1-3 次新揭秘，不能一次性全说完。

**例句（凡尘纪第 14 章）**：
> 李隙（看着赵叔）："你爹进的那口井，不是塌方——是凡尘派的地下遗迹。"  
> 李隙（握紧残简）："……他们想让他带路。"  
> 赵叔（点头）："他想毁了那里的东西，免得被他们利用。"

**机制 2：加速成长型（T0/T1）**
- **核心**：主角有"外挂老师/外挂系统"加速修炼
- **爽点构建**：别人一年突破，主角一月突破
- **代表作**：
  - 斗破 KZ-002：药老戒指
  - 大奉打更人 许七安：地书碎片
  - 雪中 徐凤年：北凉王资源
- **写法要诀**：老师必须"有缺陷"——不能万能；系统必须"有代价"——不能无限兑换。
- **必配限制**：老师要"消逝"（药老后期消逝）、系统要"过气"（后期主角不再依赖系统）。

**例句（斗破苍穹）**：
> 药老：小子，那群人看不上的东西，恰恰是最值钱的——为师教你个法子。

**机制 3：天赋血脉型（T0/T1）**
- **核心**：主角天赋异禀/血脉独特
- **爽点构建**：越级挑战、同阶无敌
- **代表作**：
  - 斗罗 KZ-003：唐三双生武魂
  - 盘龙 林雷：龙血战士
  - 凡尘纪 KZ-008：李隙双生（尘根+隙脉）
- **写法要诀**：天赋必须"先被压抑后被揭露"——开篇被嘲笑为废物，中段揭露真身。
- **必配限制**：天赋不是万能的，必须有"代价"（兽化、失控、寿命）。

**例句（斗罗大陆）**：
> 素云涛：你的武魂是蓝银草，先天满魂力……可惜是废武魂。  
> （第 N 章后）蓝银皇觉醒——唐三在众人震惊的目光中显化蓝银皇真身。

**机制 4：可进化功法（T0）**
- **核心**：主角有一部可进化的功法，从低阶一路升到天阶
- **爽点构建**：每次进化都伴随新斗技+新能力
- **代表作**：
  - 斗破 KZ-004：焚决（黄阶低级→天阶高级）
  - 凡尘纪 KZ-001 信息差 + 凡尘诀八篇（残简→起源篇→开天篇）
- **写法要诀**：进化必须有"显化场景"——不能悄悄升级，要有"突破瓶颈"的画面。
- **必配限制**：每卷最多进化 1-2 次，不可跳过中间阶。

**机制 5：越级吸收（T0/T1）**
- **核心**：主角能吸收比自己境界高的资源
- **爽点构建**：对手震惊+反杀
- **代表作**：
  - 斗罗 唐三：40 级吸收万年魂环、70 级吸收百万年魂环
  - 斗破 萧炎：斗师吞异火
  - 凡尘纪 KZ-006：越级承载命纹
- **写法要诀**：越级必须有"灵魂修为"作为硬门槛——不能无脑越级。
- **必配限制**：越级后必须有"短期虚弱期"——不能让主角吞完立刻满血。

**机制 6：有限回档（T1）**
- **核心**：主角有"有限次数"的时间回溯能力
- **爽点构建**：用信息差破局+显化代价
- **代表作**：
  - 凡尘纪 KZ-005：每卷一次，每次折寿/失忆
  - 《日食之影》：循环中改变过去
  - 《从姑获鸟开始》：单次回档
- **写法要诀**：回档必须"显化代价"——头发白、视力降、寿命减。
- **必配限制**：全书 5-7 次封顶，每卷 1 次，不可滥用。

**机制 7：前世记忆（T1）**
- **核心**：主角带着前世知识穿越
- **爽点构建**：用前世知识碾压当世
- **代表作**：
  - 斗罗 KZ-007：唐三的唐门暗器知识
  - 大奉打更人：许七安的现代警察推理
  - 全球高武：方平的前世记忆
- **写法要诀**：前世知识要"克制"——前期多，后期少（因为本世已经超越前世）。
- **必配限制**：前世知识不能"全知"——必须有"本世规则不同"的修正。

**机制 8：体制/身份背书型（T1/T2）**
- **核心**：主角的特殊身份（穿越者、上交者、被选中者）
- **爽点构建**：身份带来的"制度性优势"
- **代表作**：
  - 上交国家流：主角是"国家的人"
  - 雪中 北凉王：身份是制度性背书
  - 玄鉴仙族：家族继承+制度权力
- **写法要诀**：身份背书要让读者"信服"——必须有"获得身份"的过程，不能空降。
- **必配限制**：身份不能"无敌"——必须有身份带来的"道德负担"（参考凡尘纪"凡尘公学讲师"的身份）。

#### 2.2.4 金手指 5 大禁忌

1. **金手指不能太万能**——没有限制的金手指 = 没有张力。
2. **金手指不能太早揭示全部**——必须"分批释放"。
3. **金手指不能完全靠老师/系统**——主角必须有"自己的判断"。
4. **金手指不能让主角失去人性**——神级金手指必须配"代价是人性"。
5. **金手指不能照搬原作**——必须有"本土化变形"（凡尘纪的"信息差"是"反血统论内核"，与斗破的"戒指老爷爷"完全不同）。

### 2.3 十架构型谱（10 大故事结构）

"十架构"是本方法论提出的**网文故事的十种主流宏观结构**。每种结构对应不同的爽点密度曲线与情绪曲线。

#### 架构 1：升级打怪型（最经典）

```
结构：每卷一张新地图，每张地图一个新反派，反派等级比主角高 1-2 个大境界
调性：热血、燃、目标明确
代表：斗破苍穹、凡人修仙传
爽点曲线：
  每 30 章 1 个小高潮（越级反杀）
  每 100 章 1 个大高潮（地图突破）
  每 200-300 章 1 个史诗高潮（地图+境界双突破）
适用：A1/A2/A5/C3
```

**例句（斗破）**：
> 第一卷：加玛帝国（斗者-斗师）  
> 第二卷：迦南学院（斗师-斗灵）  
> 第三卷：云岚宗+黑角域（斗灵-斗皇）  
> 第四卷：中州（斗皇-斗宗-斗尊）  
> 第五卷：古界（斗尊-斗圣）  
> 第六卷：魂族决战（斗圣-斗帝）

#### 架构 2：学院派（最易上手）

```
结构：主角进入某学院→ 7-9 个伙伴成团→ 学院会战→ 离校
调性：青春、热血、群像
代表：斗罗大陆、大奉打更人（部分）
爽点曲线：
  开篇 30 章：伙伴集结
  30-100 章：日常训练+小冲突
  100-200 章：学院会战（大赛）
  200+ 章：离校
适用：A2/C4
```

**例句（斗罗）**：
> 史莱克七怪：戴沐白（邪眸白虎）、奥斯卡（香肠）、唐三（蓝银草+昊天锤）、小舞（柔骨兔）、马红俊（邪火凤凰）、宁荣荣（七宝琉璃塔）、朱竹清（幽冥灵猫）

#### 架构 3：地图递进型（爽点密集）

```
结构：第 1-30 章矿区 → 30-60 章城市 → 60-90 章宗门 → 90-120 章名门 → 120+ 远古
调性：开图爽
代表：凡尘纪、斗破、盘龙
爽点曲线：每 30 章换地图，每张地图有新资源+新敌人
适用：A1/A2/C3
```

**例句（凡尘纪）**：
> 卷一：黑水镇矿区（感尘境）  
> 卷二：砾川城（纳气境）  
> 卷三：万纹森林+尘野院（凝轮境）  
> 卷四：七星道统（化相境）  
> 卷五：四域（洞天境）  
> 卷六：归墟岛+八遗族（归一境）  
> 卷七：天界+司天神庭（渡厄-开天）

#### 架构 4：副本递进型（节奏最稳）

```
结构：第 1 关→第 2 关→…→第 9 关，每关是独立副本
调性：闯关、可量化
代表：海神九考（斗罗）、惊悚乐园剧本
爽点曲线：每关 1 个大爽点，9 关 9 个大爽点 + 1 个终极爽点
适用：A5/D3
```

**例句（斗罗海神九考）**：
> 第一考：穿越环形海  
> 第二考：突破海妄魔潮  
> 第三考：对抗魔魂大白鲨  
> 第四考：潮汐练体  
> 第五考：拔出海神三叉戟  
> 第六考：与七圣柱斗罗对话  
> 第七考：与深海魔鲸王战斗  
> 第八考：与自己决斗  
> 第九考：海神传承

#### 架构 5：双线并行型（群像最强）

```
结构：主角线 + 副主角线交替推进，每 100 章交叉
调性：群像、多视角
代表：雪中悍刀行、诡秘之主
爽点曲线：双主角各自爽点 + 配合爽点
适用：C1/F1
```

**例句（雪中）**：
> 徐凤年线（北凉世子→北凉王） + 徐骁线（北凉王→为子铺路） + 老黄线（老仆→独立） + 姜泥线（亡国公主→独立）  
> 4 条线交替推进，每 50 章交叉一次

#### 架构 6：推理破案型（最烧脑）

```
结构：每个案件独立成篇，案件间有大线索串联
调性：烧脑、揭露
代表：大奉打更人、惊悚乐园（部分）
爽点曲线：每案 1 个大爽点（指认真凶），每 3-5 案 1 个大线索揭露
适用：C4/D1/D2
```

**例句（大奉打更人）**：
> 税银案（金属钠调包术）→ 桑泊案（永镇山河庙爆炸+平阳郡主失踪）→ 楚州血屠三千里（镇北王屠城）→ 云州叛军（弑君之战）

#### 架构 7：系统签到型（最稳定）

```
结构：主角有系统，每天签到获得资源→ 资源积累→ 兑换技能→ 升级
调性：日常、轻松、爽
代表：大王饶命、万界点名
爽点曲线：每章小爽点（签到奖励），每 50 章大爽点（系统升级）
适用：B2
```

**例句（大王饶命）**：
> 吕树负面情绪收集系统  
> 每天毒舌→ 收集负面情绪→ 抽奖（洗髓果实、扭头葫芦、天下潮、世界树）  
> 50 万字 → 系统升级 → 负面情绪值破亿 → 一品

#### 架构 8：家族经营型（最慢热但最稳）

```
结构：主角是家族继承人→ 经营家族产业→ 培养族中人才→ 抵御外敌
调性：经营、群像、长线
代表：玄鉴仙族、家族修仙
爽点曲线：每 30 章家族升级，每 100 章外敌入侵
适用：C2
```

**例句（玄鉴仙族）**：
> 季越人版社畜魂穿器灵→ 经营家族灵田→ 培养族中子弟→ 应对天灾、敌对家族、神祇

#### 架构 9：多主角群像型（最考验功力）

```
结构：3-5 个主角视角，每视角 1-3 个核心人物
调性：群像、厚重
代表：雪中悍刀行、诡秘之主（克莱恩+奥黛丽+阿尔杰）
爽点曲线：每视角独立爽点+大事件协同
适用：C1/F1
```

**例句（诡秘之主）**：
> 克莱恩（主角，序列 0 诡秘之主）  
> 阿尔杰（海王，序列 4 灾难主祭）  
> 奥黛丽（观众，序列 4 神秘学家）  
> 3 视角推进，每 100 章协同一次

#### 架构 10：反英雄型（最有争议）

```
结构：主角是反派/灰色人物，按自己的逻辑行事
调性：偏邪、灰、争议
代表：亵渎、诛仙张小凡（部分）、雪中部分角色
爽点曲线：与正派对抗+内心戏
适用：A4/C1
```

**例句（亵渎）**：
> 罗格是杀手+法师+主教，灰色地带  
> 主角按自己逻辑行事，不被道德绑架  
> 读者前期"恨"主角，后期"理解"主角

#### 架构 11（凡尘纪专属）：制度批判型

```
结构：主角的敌人不是某个人，而是某套"制度"
调性：沉重、史诗、反血统论
代表：凡尘纪
爽点曲线：每卷揭露一层制度黑幕，每 3 卷推翻一层
适用：A1+C1+A4 复合
```

**例句（凡尘纪）**：

> 卷二：揭露散修城市的天网监控

#### 资源点使用 5 大规则

1. **分批投放**——每卷 2-3 种新资源，不可一次用尽。
2. **主角+盟友共分**——主角独占 = 烂尾风险高。
3. **部分公共化**——资源 30% 必须公共化（凡尘纪的"凡尘公典"）。
4. **代价显化**——每次获取资源必须有"代价"（受伤、消耗、伦理冲突）。
5. **废弃品也要有名**——失败的资源也要"露脸"，增加真实感。

### 3.3 鬏点主调（5 大鬏点模式 + 配比）

"鬏点"是主角的"标志性时刻"——每个主角必须有 1-2 个"鬏点"，是读者记住这个主角的核心。

#### 5 大鬏点模式

```
鬏点 1：觉醒鬏点
  形态：主角从废物变强的那一刻
  代表：萧炎药老觉醒、唐三双生武魂觉醒
  写法要诀：必须"慢镜头"+ "内心独白"
  例句：
    > （萧炎闭目）"三十年河东，三十年河西——莫欺少年穷。"
  使用频次：1 次/书（开篇）

鬏点 2：拒绝鬏点
  形态：主角拒绝某种诱惑（不称王、不称霸）
  代表：李隙拒绝继任司天君
  写法要诀：必须"长台词" + "全场静默"
  例句：
    > "我不是你们的王，也不是你们的神。我要做的事只有一件——把门打开。"
  使用频次：1 次/书（终卷）

鬏点 3：告白/承诺鬏点
  形态：主角对重要的人/事许下承诺
  代表：药老承诺"为师教你个法子"、李隙承诺"还愿"
  写法要诀：必须"简短有力"+ "可被验证"
  例句：
    > 李隙："我不是为父报仇，是替父还愿。"
  使用频次：1-2 次/书

鬏点 4：牺牲鬏点
  形态：主角为他人牺牲的时刻
  代表：唐昊献祭、铁铮盾碎矿心
  写法要诀：必须"慢镜头" + "遗言"
  例句：
    > 铁铮（最后一句）："告诉孩子们，盾是用来挡风的，不是挡命的。但今天，先挡命。"
  使用频次：1-3 次/书

鬏点 5：真相鬏点
  形态：主角揭露/见证重大真相
  代表：李隙揭露废根档案、萧炎揭露魂族阴谋
  写法要诀：必须"具象化"——不能只说"我发现了真相"，要展示"档案""证据""证人"
  例句：
    > 守简人族长（发现废根档案）："我祖上也曾是'被废'。"
  使用频次：1-2 次/卷
```

## 第四部分：五层场景 / 七层地图 / 十拓扑

### 4.1 五层场景（章节级场景结构）

任何一章 3000-3500 字的网文，都应该遵循 **五层场景结构**：

```
层 1：日常层（300-500 字）
  目的：让读者进入"这一章的世界"
  技法：环境描写+人物状态+小事件
  例句：
    > 黑水镇晨雾。李隙与铁子等十几名星奴在九号矿井口列队，监工刘三举着铜鞭数人头。

层 2：冲突层（500-800 字）
  目的：建立本章的"小冲突"
  技法：人物之间的张力+主角的挑战
  例句：
    > 刘三以"产量不达标"为由，将李隙三日口粮划掉。铁子想分自己半块饼给李隙，被刘三一鞭抽开。

层 3：高潮层（800-1200 字）
  目的：本章的"小爽点"或"小压抑"
  技法：情节推进+情绪爆发
  例句：
    > 李隙在矿壁凿痕处用气感凝砂抵住巨石，救了铁子——代价是左臂经脉拉伤。

层 4：情绪层（500-800 字）
  目的：让读者"共情"主角
  技法：内心独白+人物关系
  例句：
    > 铁子（拉住他）："我替你瞒。"李隙（捂住左臂）："别说话——快回去。"

层 5：钩子层（300-500 字）
  目的：让读者"不得不看下一章"
  技法：悬念/反转/预告
  例句：
    > 这一幕被躲在暗处的刘三看见了——他去举报。
```

层 7：天界/终极（卷 7，715-833 章）

### 4.3 十拓扑（10 大势力/组织架构）

"十拓扑"是 **势力架构的 10 种主流结构**。每本玄幻爽文必须有 1-2 个"主拓扑"。

```
拓扑 1：中央集权型
  形态：1 个中央 + N 个分部
  例：武魂殿（斗罗）、魂殿（斗破）
  适用：A1/A2
  写法：中央反派是长线 BOSS

  例：七星道统（凡尘纪）、八古族（斗破）
  适用：A4/C1

拓扑 3：帝国/王朝型
  形态：1 个帝国 + 多级贵族
  例：加玛帝国（斗破）、大奉王朝（大奉打更人）
  适用：A1/F1
  写法：皇帝是隐藏 BOSS，贵族是中期 BOSS

拓扑 4：学院/门派型

  例：史莱克学院（斗罗）、迦南学院（斗破）

  适用：C2/A1

拓扑 6：神祇/天界型
  形态：1 个天庭 + N 个神殿
  例：神庭（凡尘纪）、神界（斗罗）

  写法：天庭是终极 BOSS

拓扑 7：禁兽/异族型
  形态：1 个异族 + N 个氏族
  例：太虚古龙族（斗破）、禁兽谱（凡尘纪）
  适用：A5/E1
  写法：异族是盟友或对手

拓扑 8：副本/秘境型
  形态：N 个独立副本
  例：海神九考（斗罗）、惊悚乐园剧本（惊悚乐园）

拓扑 9：穿越/系统型

### 5.2 修正位（10 个常见 bug 修正）

"修正位"是 **"金手指失效的常见场景"及"修正方法"**。

```
修正位 1：金手指太万能
  场景：主角用金手指无成本解决一切
  修正：每用一次金手指必须付出代价
  例：凡尘纪每卷一次回档，代价显化

修正位 2：金手指太早揭示全部
  场景：金手指一开篇就展示全部能力
  修正：分卷释放，每卷 1-3 次新能力
  例：凡尘纪第一卷只用残简感尘篇，第二卷才升级到纳气篇

修正位 3：金手指全靠老师/系统
  场景：主角全靠老师/系统拿主意
  修正：关键时刻主角必须有"自己的判断"
  例：李隙在凡尘纪中后期拒绝"按父亲实验走"

修正位 4：金手指让人性丧失
  场景：主角太强而失去人性
  修正：金手指必须有"代价是人性"
  例：凡尘纪回档代价包含"记忆模糊"

修正位 5：金手指泛滥到烂尾
  场景：金手指从"震撼"变"平淡"
  修正：金手指必须有"失效条件"
  例：凡尘纪 KZ-005 有限回档有"次数上限"

修正位 6：伙伴全成挂件
  场景：7 个伙伴只为主角服务
  修正：每卷至少 1-2 场"无主角解局"
  例：凡尘纪第 20 章七契争议、姜烬自述

修正位 7：资源堆积无意义
  场景：主角一路捡资源但不用
  修正：每卷必须"用掉"1-2 个资源
  例：凡尘纪每卷用 1-2 种奇质

修正位 8：制度批判只喊口号
  场景：主角批评制度但没方案
  修正：批判必须配"建设性方案"
  例：凡尘纪"反血统论"必须配"凡尘公典"+"凡尘公议"+"凡尘公学"

修正位 9：反派降智
  场景：反派一路送死
  修正：反派必须有"自己的逻辑"
  例：凡尘纪司天君的"资源有限论"是有逻辑的

修正位 10：终局空喊自由
  场景：主角推翻后没有方案
  修正：终局必须配"具体制度"
  例：凡尘纪"公议+公典+校验"三制度
```

## 第六部分：修炼体系与战力设计

### 6.1 修炼体系设计的 4 大原则

```
原则 1：可量化
  - 境界必须有数字
  - 例：斗破斗气 1-9 段+斗者 1-9 星+…
  - 优势：让读者"有数字感"
  - 风险：后期数字爆炸（斗圣九转×3=27 个小阶）

原则 2：可突破
  - 每个境界都有"突破感"
  - 例：突破时有"天象变化"
  - 优势：让读者"有爽点"
  - 风险：突破太频繁会"廉价"

原则 3：可对比
  - 同阶之间有"层次"
  - 例：斗破斗师九星 vs 斗师一星
  - 优势：让读者"看得懂"
  - 风险：复杂到读者记不住

原则 4：可控制
  - 后期不能靠数字压制
  - 必须靠"领域"、"法则"、"权限"
  - 例：凡尘纪归一境后是"法则权限"
  - 优势：避免"战力崩坏"
  - 风险：早期没设计好，后期会崩
```

### 6.2 修炼体系模板（6 大主流体系）

```
体系 1：单线量化型
  例：斗破斗气 11 大阶 1-9 段
  优势：易理解
  风险：后期数字爆炸

体系 2：多线技能型
  例：斗罗魂师等级+魂环+魂技+魂骨
  优势：维度多
  风险：体系复杂

体系 3：多线复合型
  例：凡尘纪修境+命纹+奇质三轨并行
  优势：组合多
  风险：需要细纲支持

体系 4：多分支型
  例：大奉打更人 10 大修炼体系
  优势：多样性
  风险：体系过于复杂

体系 5：规则数值型
  例：惊悚乐园 F-E-D-C-B-A-S 等级
  优势：易量化
  风险：缺乏浪漫感

体系 6：血脉觉醒型
  例：盘龙 龙血战士
  优势：单线突出
  风险：容易"血统论"
```

### 6.3 战力设计的 5 大规则

```
规则 1：跨阶挑战必须有 4 个条件
  - 灵魂修为 > 对手（神识压制）
  - 战术/阵法 > 对手（环境压制）
  - 资源/装备 > 对手（物质压制）
  - 意志/情感 > 对手（精神压制）
  例：凡尘纪越级吸收命纹需"肉身/神识/阵辅/意志"四条件

规则 2：每战必有三层反馈
  - 战前：铺垫
  - 战中：高潮
  - 战后：影响（伤亡/政治/资源）
  例：凡尘纪第 30 章矿心冷焰战+蓝光冲天+卷终政治影响

规则 3：必须有"公共阵网"机制
  - 主角不能独大
  - 后期必须有"众人合力"机制
  例：凡尘纪万民灯阵

规则 4：高境不可被低境无脑堆死
  - 跨大境界必须靠"领域/法则"
  - 不能靠"人海战术"
  例：凡尘纪"高境修士强在领域和权限"

规则 5：资源数量不直接等于胜负
  - 兼容度+组合方式更重要
  - 例：凡尘纪"奇质不直接等于胜负"
```

## 第七部分：细纲与扩写手法

### 7.1 细纲结构（4 段式）

每章 3000-3500 字的细纲，必须包含 4 段：

```
段 1：章目标题 + 一句话核心冲突
  例：第 11 章 灰甲噬晶兽——首战禁兽·断灵首秀

段 2：本章目标 + 信息增量
  例：
    本章目标：李隙第一场对敌战斗——对象是矿区深处的灰甲噬晶兽。
    信息增量：
    - 灰甲噬晶兽：矿区常见的灰阶禁兽
    - 断灵技巧：凡尘诀基础战技
    - 命纹前置：可提取"灰纹"

段 3：幕表（5 幕）+ 关键对话
  例：
    幕 1（200 字）：兽吼
    幕 2（200 字）：监工下令撤出
    幕 3（200 字）：遭遇
    幕 4（200 字）：断灵
    幕 5（200 字）：击杀
    关键对话：李隙（按住兽颌）："断灵——"

段 4：爽点/钩子设计 + 接入前章/推下章
  例：
    爽点调性：SH-004 越级反杀·首秀
    章末钩子：李隙从噬晶兽尸体的甲壳下发现一颗淡灰色的"命核"
```

### 7.2 扩写 6 步法则（增裁 / 调转 / 秩掌 / 加金 / 变压 / 复品）

扩写 = 把细纲变成正文。本方法论推荐 **6 步扩写法**：

```
步 1：增裁（增字数 + 删废话）
  - 目标：每章 3000-3500 字
  - 增什么：环境描写、人物动作、内心独白
  - 删什么：重复对话、冗余说明
  例：把"监工刘三当众羞辱李隙"扩成 200 字描写

步 2：调转（调整节奏 + 转折点）
  - 目标：每章必须有 1 个转折
  - 技法：把"直叙"改成"先抑后扬"
  例：把"李隙去找赵叔"改成"李隙先遇到刘三→再找赵叔"

步 3：秩掌（梳理节奏 + 把控画面）
  - 目标：每幕 500-800 字，画面感强
  - 技法：动作分镜、慢镜头
  例：把"李隙用断灵打噬晶兽"分成 4-5 个动作镜头

步 4：加金（加金句 + 加信息量）
  - 目标：每章 1-2 个"金句"
  - 技法：把"意思"凝成"金句"
  例：把"李隙发现噬晶兽可以杀"凝成"断灵——"

步 5：变压（变化情绪压力 + 张力）
  - 目标：每章情绪曲线起伏
  - 技法：紧张→松弛→紧张
  例：第 11 章紧张（兽吼）→松（监工撤）→紧张（遭遇）→爽（断灵）→松（击杀）

步 6：复品（复查质量 + 主题校验）
  - 目标：每章完成后做"主题校验"
  - 校验项：是否兑现前章钩子？是否埋下章伏笔？是否体现主角人设？是否服务反血统论？
```

### 7.3 扩写 6 步实例

**例：把"李隙用断灵打噬晶兽"扩写成 600 字正文**

## 第八部分：创作不可质越 · 9 大禁忌

本方法论总结 **9 大禁忌**——任何一条触碰，都可能让书烂尾或被读者抛弃。

### 禁忌 1：主角人设漂移

```
场景：开篇主角"冷静克制"，中段突然"热血冲动"
后果：读者觉得"换主角了"
修正：每 30 章做一次"主角声音"校验
```

### 禁忌 2：金手指泛滥

```
场景：金手指一开篇就展示全部能力
后果：后期"无新东西"
修正：分卷释放，每卷 1-3 次新能力
```

### 禁忌 3：资源堆积无意义

```
场景：主角一路捡资源但不用
后果：资源"通货膨胀"
修正：每卷必须"用掉"1-2 个资源
```

### 禁忌 4：伙伴全成挂件

```
场景：7 个伙伴只为主角服务
后果：群像戏失败，读者觉得"这团队假"
修正：每卷至少 1-2 场"无主角解局"
```

### 禁忌 5：制度批判只喊口号

```
场景：主角批评制度但没方案
后果：读者觉得"主角空喊"
修正：批判必须配"建设性方案"
```

### 禁忌 6：反派降智

```
场景：反派一路送死
后果：胜利感降低，读者觉得"主角是傻子"
修正：反派必须有"自己的逻辑"
```

### 禁忌 7：终局空喊自由

```
场景：主角推翻后没有方案
后果：烂尾感强烈
修正：终局必须配"具体制度"
```

### 禁忌 8：照搬原作 30 字以上

```
场景：直接抄原作对白/情节
后果：被读者发现，评分暴跌；可能涉及版权
修正：所有对白/情节必须"原创"或"变形"
```

### 禁忌 9：凡人大众消失

```
场景：主角与"最底层"脱节
后果：失去"代入感"——读者就是"最底层"
修正：凡人大众必须每 5 章出现一次
例：凡尘纪浮萍每 5 章一次
```

def check_compatibility(card_db, selected_ids):
    """检查所选卡片的兼容性"""
    selected_cards = []
    for cid in selected_ids:
        card, _ = find_card_by_id(card_db, cid)
        if card:
            selected_cards.append(card)
    
    score = 0
    for card in selected_cards:
        for compat in card.get('compatible_with', []):
            if compat in selected_ids:
                score += 1
    return score, len(selected_ids)

# 示例：检查凡尘纪原版配方

])
