## 一、作品简介

如果有一天，太阳消失了，人间一片黑暗，会是什么样的世界呢？

巨变中的地球陷入了黑暗，从此没有了阳光，没有了星空，只有无尽的寒冷与黑夜，人类从此进入黑暗与血的时代……

《黑暗血时代》是末世题材的重要里程碑，被读者评价为"最有逻辑且场面宏大的网络小说之一"。作者天下飘火凭借此书成为阅文集团知名作家，2014年获得科幻灵异之王奖。截至2026年，VIP字数满20W字，均订数据大于1W，拥有100个及以上盟主级别粉丝。

**核心看点**：太阳消失引发的全球灾难→人类觉醒成为黑暗武士→修炼体系与符文科技→星际文明冲突→维度战争

## 二、核心世界观

### 2.1 末世设定

**时间起点**：2012年12月22日  
**触发事件**：太阳在无任何征兆情况下消失10分钟（北京时间7:15），引发全球混乱与恐慌。该事件验证了玛雅预言，引发世界各地骚乱（纽约皇后区、巴黎爱丽舍宫等），人类开始认识到末世降临。

**末世影响**：
- 地球陷入无尽黑暗与寒冷
- 神秘生物降临地球（虫族/珉）
- 人类觉醒成为"黑暗武士"（简称黑武）
- 失去阳光后的人类文明重建

### 2.2 宇宙观

| 概念 | 说明 |
|------|------|
| **暗能量** | 第四维能量，宇宙生命天生拥有并能使用暗能量，地球是被封锁的三维空间 |
| **暗域** | 宇宙中充斥暗能量和暗物质的区域，无物质，资源匮乏，危机四伏 |
| **气泡世界** | 白茫茫无颜色无空间的世界，气泡=生命的零维世界，如潮汐般起伏泯灭 |
| **彩虹桥** | 神战遗址，崩塌的神秘通道 |
| **节点** | 宇宙推演模型，建于比最小尺度还小的破碎间隙 |
| **降维降临** | 高级外星种族进入地球，由四维进入三维，会被宇宙抹掉多出的维度 |

### 2.3 地球七纪

小说时间跨度跨越地球纪元的七个阶段，从原始到星际文明的发展历程：
- **第一纪~第六纪**：人类文明演进
- **第七纪**：当前故事发生的主要时代

## 三、修炼体系详解

### 3.1 基础架构

**核心概念**：
- **黑暗能量**（古书称"天地元气"）：主角和一般黑武拥有的能量
- **零维空间**：所有生命意识存在空间，时间轴存在=活着，时间轴消失=死亡
- **契约**：生命进入"枢机"境界的分水岭，用于巩固零维

**意识层级**：
1. 第一限级：分辨世界真假
2. 第二限级：分辨自己真假
3. 第三限级：分辨生死真假

### 3.2 修炼境界（九元天）

| 境界 | 等级名称 | 特征 | 战力 |
|------|----------|------|------|
| **一元天** | 储元体 | 重复吸收释放元气淬炼身体，积累九九八十一次后达巅峰 | 战力平平，无太多战斗力 |
| **二元天** | 融元体 | 元气与身体融合，完成五层融元体修炼 | 元气量显著增加，本体元气更强大 |
| **三元天** | 纯元体 | 分底层、中层、上层三层，缓缓将本体改造成纯元体 | 元气可外放形成，可任意转变形态如元气剑 |
| **四元天** | 本源体 | 修炼自身源体，本体元气形成本源元气，身体改造为本源源体 | 可掌控天地间大量暗能量，动辄毁灭一座城池，掌握"枢机之力" |
| **五元天** | 命源体 | 修炼命源 | 战力较四元提升显著 |
| **六元天** | 合源体 | 将本源体与命源体融合形成完整的合源体 | 巅峰枢机战力提升幅度巨大，一人可对付数个低阶枢机 |
| **七元天** | 一阶形态源门 | 可形成"轭"场（源空之地），掌握一片区域的暗能量 | 念动间束缚或致命打击敌人，残留一片也能存活 |
| **八元天** | 二阶形态源门 | 额外掌握"击"之法——纯粹攻击战法 | 战斗力极其强大，贯穿数颗普通恒星不在话下，可将本体分裂为多个源体组织 |
| **九元天** | 巅峰源门 | 源门生命最巅峰 | 足以傲视其他非巅峰源门 |

### 3.3 灵生命体系（最高境界）

**诞灵条件**：九元天大成，修炼意识原体，进而诞灵

**灵之位级（7个显著境界）**：

| 位级 | 说明 |
|------|------|
| 新灵 | 新诞生的灵生命 |
| 初灵 | 初级灵生命 |
| 上位之灵 | 比初灵强大很多 |
| 迷之境界 | 初灵晋升2个灵位后，可在多维空间中寄存存活 |
| 灵尊 | 普通灵对极其强大灵的一种尊称与概括 |
| 顶级灵体 | 如七钉继承人、八域巡天使 |
| 巅峰之灵 | 最强境界，如破镇之人、冥、左旋新神尊 |

**灵的三大通用能力**：
- **无形**：在不同生命眼中呈现为不同生命所认为的"神"的形象
- **无音**：说话直接反应在不同生命的零维空间中，跨越语言问题
- **无损**：降临各种生命体，离开时不造成任何伤害

**基础通用型灵法——时空径迹**：用灵蕴拉开一片时空间，将世界的变化缓慢为一帧一帧的"动态3D图片"，灵可在世界面上进行干涉，抹去任意物体。

## 四、符文科技体系

### 4.1 元符结构

元符由三个部分组成：
- **符首**：引导天地法则的通道，将天地元气引入符本体的必经之路
- **本体**：引入元气后，通过规则方式建立某种能力的法则，实现神通
- **符封**：激活本体功能后封印元符，印上制符人气息或制作成无主元符

### 4.2 元符分类

| 分类 | 说明 |
|------|------|
| **攻击元符** | 需要利用自身元气炼制，使用时瞬发，可在神域中凌空篆符 |
| **加成元符** | 封印在人体或装备上起到加成作用 |
| **辅助元符** | 非战斗类，只起辅助作用 |
| **一次性元符** | 二阶及以下，每次元符能量不足需更换新的 |

### 4.3 符箓用途一览（节选）

| 符箓名 | 等级 | 属性 | 用途 |
|--------|------|------|------|
| 火兵符 | - | 火 | 改造武器运用天地元气 |
| 寒元符 | - | 冰 | 改造武器运用天地元气 |
| 摄元符 | 二阶/三阶 | 无 | 吸收储存改造后的元气，三阶可循环使用 |
| 聚元符 | - | 无 | 自动摄取天地元气，补充武器或入体 |
| 愈宫符 | 三阶 | 无 | 疗伤活命 |
| 驱毒符 | 三阶 | 无 | 去除不同属性毒素 |
| 封兽符 | 四阶 | 无 | 封印拘禁濒死怪物入符体，可保存被封印怪物意识 |
| 离火符 | 三阶 | 火 | 催发神火 |
| 冰崩符 | 三阶 | 冰 | 形成冰川包裹敌人然后崩裂 |
| 土御符 | 三阶 | 土 | 大大增加防御力 |
| 通壁符 | - | 无 | 帮助一元天修炼者开辟与天地元气沟通的通道 |
| 天地元符 | 四阶 | 无 | 将元气摄入体内，冲刷意识神识，打开第四维 |

### 6.2 大剧情阶段

根据读者整理，《黑暗血时代》剧情可分为三个主要阶段：

#### 阶段一：地球篇（第1章~第1000章）

**主要副本与剧情进展**：

**【迷雾之城篇】**
- 楚云升在城市废墟中挣扎求存
- 遭遇各种怪物（虫族）袭击
- 与其他幸存者组成临时团队
- 获得家传古书《神储诏书》，开始修炼

**【金陵城篇】**
- 建立或加入金陵城势力
- 势力纷争与资源争夺
- 黑武（黑暗武士）觉醒者的崛起
- 与其他势力的冲突与博弈

**【黄山篇】**
- 黄山决战——重要战役
- 楚云升誓言不降的标志性场景
- 势力整合与联盟形成

**【决战港城篇】**
- 大规模战争场面
- 多方势力混战
- 楚云升的关键战斗

**【蜀都一剑篇】**
- 标志性战斗场景
- 楚云升实力展现

#### 阶段二：节点篇/火星篇

- 离开地球，进入更广阔的星际空间
- 遭遇卓尔人等外星文明
- 楚云升实力的进一步提升
- 新的敌人和挑战

#### 阶段三：星空篇

- 进入真正的星际舞台
- 左旋神国与新神国的冲突
- 宏科技与符文科技的交织
- 天外邪魔逼近的威胁
- 楚云升成长为宇宙级强者

## 七、主角详解

### 7.1 楚云升

**身份**：被称为"典主"、"虫典之主"

**修炼层次**：
- 本体：初入六元天
- 融合生命虫甲后：在七元天境界有一段距离

**核心能力**：

| 能力 | 说明 |
|------|------|
| **种子（意识原体）** | 可产生第三股力量，带有强烈原始欲望。可被坚强意志压制，突破第二限级后可从零维内看到种子 |
| **黑气漩涡** | 封兽符逆转后因情绪剧烈波动产生，具有维护零维空间的力量 |
| **黑气** | 具有追本溯源能力，可将其他物质归还为最根本的"真空"，无视维度差，可自由穿梭 |
| **物子碎片** | 炁始祖遗物，融合意识后可极快出体攻击 |
| **立方体** | 宏领域科技产物，具有抵抗灵级信息量及笼罩保护零维能力 |

**武器装备**：

| 武器 | 类型 | 状态 |

**战技**：

| 类型 | 战技 |

- 一般黑武只拥有一种或两种黑暗能量，应用方法有限，没有功法难以升级

- 主角元气属于超稳定性，随元天增加越发稳定
- 主角能在源门源空之地范围内释放攻击

### 10.1 核心爽点类型

**1. 绝境逆袭型**
- 面对远超自身的敌人，通过智慧和技巧反杀
- 代表场景：楚云升以弱胜强的多场战斗

**2. 修炼突破型**
- 苦修后的实力跃升，一鸣惊人
- 古书修炼体系的独特性带来差异化成长感

**3. 势力崛起型**
- 从孤身一人到建立势力，指挥大军
- 代表场景：金陵城势力整合、黄山决战

**4. 星际史诗型**
- 地球篇后期展开的星际舞台
- 宏科技与符文科技的碰撞

**5. 情感羁绊型**
- 楚云升与冥、宋影、小老虎的羁绊
- 无数轮回中的陪伴
