## 一、作品简介

### 1.1 故事背景

这是一个广博的魔幻世界。强者可以站在黑色巨龙的头顶遨游天际，恐怖的魔法可以焚烧江河，可以毁灭城池，可以夷平山岳……这里无奇不有。

大陆上传说中的四大终极战士之一的"龙血战士"已经千年没有再出现过了，而唯一有着龙血战士血脉的家族也渐渐衰败了下来，成为了一个小镇的普通贵族。而这个衰败家族中的继承人——**林雷**，便是本书的主角。

### 1.2 核心设定

| 设定 | 说明 |
|------|------|
| 盘龙戒指 | 主角获得的神秘戒指，贯穿全书的重要道具 |
| 龙血战士 | 四大终极战士之一，千年未出现 |
| 四系魔法 | 地、水、火、风四大元素魔法 |
| 多维位面 | 物质位面、神位面、至高位面 |

### 1.3 故事主线

主角林雷出生于玉兰大陆已衰落的龙血战士家族，在偶然捡到一枚盘龙戒指后成为天才魔法师，引动龙血战士血脉，振兴家族，一路成神，最终成为第二位**鸿蒙掌控者**。

- 前期被看不起的家族继承人，最终成为鸿蒙掌控者

## 八、写作特色

1. **海外传播成功**：被译介到海外，成为网络文学海外传播的标杆
