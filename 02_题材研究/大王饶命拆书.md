# 《大王饶命》详细拆书报告

> 作者：会说话的肘子 | 总字数：约300万字 | 完结时间：2018年 | 题材：都市异能/灵气复苏

---

## 一、作品定位与核心设定

### 1.1 题材类型
都市灵气复苏流+系统吐槽流。主角吕树以"毒舌"为核心，通过收集他人负面情绪值换取系统积分，是网文界将"嘴炮"与升级体系深度融合的标杆作品。

### 1.2 核心设定速览

| 设定项 | 内容 |
|--------|------|
| **双重世界** | 地球（灵气复苏后） ↔ 吕宙（修仙世界） |
| **地球修行等级** | F→E→D→C→B→A→神藏境（大宗师） |
| **吕宙修行等级** | 六品→五品→四品→三品→二品→一品→大宗师→两宙境→神王境 |
| **地球组织** | 天罗地网（十二天罗）、基金会 |
| **吕宙组织** | 神王宫、御龙班直、武卫军 |
| **核心金手指** | 负面情绪收集系统（抽奖、兑换果实） |
| **核心道具** | 洗髓果实、扭头葫芦、天下潮（剑）、世界树 |

### 1.3 修炼体系对照

| 地球等级 | 吕宙等级 | 能力描述 |
|----------|----------|----------|
| F级（六品） | 六品 | 初步觉醒 |
| E级（五品） | 五品 | 超现实力量 |
| D级（四品） | 四品 | 躲避热武器 |
| C级（三品） | 三品 | 对抗热武器 |
| B级（二品） | 二品 | 借用天地之力 |
| A级（一品） | 一品 | 与天地共鸣 |
| 神藏境 | 大宗师 | 地球上最强级别 |
| — | 两宙境 | 超越大宗师 |
| — | 神王境 | 吕宙最高统治者 |

## 二、势力组织架构

### 2.1 地球势力

**天罗地网**
- 第一天罗聂廷：东方第一高手，新亭侯，刀法无双，后自废根基降至一品
- 第三天罗石学晋：道士，性情暴躁，盐湖遗迹后突破为地球第一个A级
- 第四天罗吕小鱼的外护：直率但办事不靠谱，一心只想挣钱
- 十二天罗：共12人，分管不同区域
- 第六天罗风云彦：具现系觉醒者，游戏宅，后辞职
- 第七天罗风云鹿：具现系觉醒者，游戏宅，辞职能否待确认

**基金会**
- 九大理事之一李弦一：吕树邻居，剑阁传人
- 幕后势力：与神王宫旧事有关联

### 2.2 吕宙势力

**神王宫体系**
- 老神王（吕神）：吕宙统治者，被恶念害死
- 吕神恶念：谋反者，夺走暗图
- 御龙班直：神王亲卫，张卫雨、东晔等
- 武卫军：吕树在吕宙建立的军队

**傀儡师**
- 云倚、虎执、泪诀：随吕神征战三千余年的忠诚部下
- 职责：保护吕树、调查18年前真相

**血妖大军**
- 敌方主力，最终决战中与御龙班直对战

## 三、主角人物分析

### 3.1 吕树

**身份背景**
- 地球：高中生，父母失踪，与吕小鱼相依为命
- 真实身份：吕神之子（吕宙神王之子），被傀儡师护送至地球
- 系统：负面情绪收集系统，可抽奖、兑换果实

**性格特质**
- 核心：腹黑毒舌、护短、有商业头脑
- 表面：嘴贱、爱占便宜
- 骨子里：善良、有底线、不主动惹事但绝不怕事

**能力发展**
| 阶段 | 境界 | 主要事件 |
|------|------|----------|
| 初始 | F级 | 获得负面情绪系统 |
| 成长期 | E-D级 | 卖臭豆腐赚积分 |
| 崛起期 | C-B级 | 加入天罗地网，遗迹探索 |
| 飞跃期 | A级（一品） | 长白山之战，吕宙归来 |
| 巅峰期 | 神藏境（大宗师） | 感悟"爱"晋升大宗师 |
| 超越期 | 两宙境 | 返回吕宙继承王位 |

### 3.2 吕小鱼

- 身份：吕树最亲近的人，与吕树无血缘关系
- 性格：傲娇、暴力倾向、极度依赖吕树
- 能力：修炼天赋极高，石学晋想收徒被拒
- 结局：陪伴吕树到最后

## 四、每卷详细剧情进展

### 【卷一·地球篇：高中时代】

---

#### 第一卷：觉醒（1-100章左右）

**核心事件**：获得系统 → 臭豆腐生意 → 道元班入学

**剧情概述**：
高中生吕树在一场车祸中改变人生，灵气开始复苏。他获得了一个可以通过收集他人负面情绪值来抽奖和兑换的系统。

吕树以卖臭豆腐为业，利用系统抽奖获得洗髓果实等道具，一边赚钱一边提升实力。他的毒舌属性开始显现，周围人被他气得不轻，负面情绪值源源不断。

进入道元班学习，开始接触修行者世界。结识室友和同学，在校园中展开一系列有趣的互动。

**关键转折**：
- 获得负面情绪系统，这是全书的核心金手指
- 父母失踪的线索开始浮现
- 发现自己的修行天赋

**境界变化**：F级 → E级 → D级

**本卷成就**：积累初始资金和实力，建立人脉网络

---

#### 第二卷：崛起（100-200章左右）

**核心事件**：天罗地网关注 → C级觉醒者 → 遗迹探索

**剧情概述**：
吕树实力快速提升，引起天罗地网的注意。第四天罗被派往洛城外国语学校担任校长，实际是调查吕树。

吕树与天罗地网产生交集，开始执行各种任务。遗迹开始出现，吕树进入遗迹探险，获得更多资源和道具。

与聂廷、石学晋等顶级强者产生联系。

**关键转折**：
- 被天罗地网确认为C级高手
- 开始了解修行者世界的规则
- 与吕小鱼的感情线深化

**境界变化**：D级 → C级 → B级

**本卷成就**：正式成为天罗地网外围成员，在修行界站稳脚跟

---

#### 第三卷：风云（200-350章左右）

**核心事件**：长白山之战 → 吕宙通道 → 神王宫真相

**剧情概述**：
海外修行者入侵长白山，聂廷为保护地球自废根基，降至一品。吕树在这场战斗中发挥重要作用。

遗迹中发现了通往吕宙的通道。吕树得知自己真实身份：他是吕宙神王吕神之子，被傀儡师护送至地球避难。

吕宙的恶念害死了吕神，夺取了神王之位和暗图。吕树必须变强才能为父报仇。

**高潮场景**：长白山大战
- 聂廷自废根基斩杀强敌
- 卡洛儿觉醒奥丁血脉
- 吕树展现真实实力

**本卷成就**：真相揭晓，确立复仇主线

---

### 【卷二·吕宙篇：异界称王】

---

#### 第四卷：跨界（350-500章左右）

**核心事件**：进入吕宙 → 武卫军建立 → 争夺王位

**剧情概述**：
吕树通过通道进入吕宙，发现这里是一个修仙文明高度发达的世界。他在这里重塑根基，修炼到更高境界。

在张卫雨、东晔等御龙班直旧部的帮助下，吕树建立了自己的势力——武卫军。

吕宙各方势力对吕树的出现反应不一，有人支持有人反对。吕树逐步收拢旧部，积累力量。

**关键转折**：
- 在吕宙恢复并超越原有实力
- 收服御龙班直旧部
- 开始了解吕神的过往

**境界变化**：吕宙一品 → 大宗师

**本卷成就**：建立武卫军，确立吕宙根据地

---

#### 第五卷：争锋（500-650章左右）

**核心事件**：王城对决 → 血妖战争 → 世界树

**剧情概述**：
吕树实力不断增强，开始与吕宙各方势力争锋。他遇到了一系列强敌，包括血妖大军等。

卡洛儿为复仇单枪匹马进攻神集，险些丧命。吕树将世界树植入她体内救她，但卡洛儿记忆被封印。

吕树感悟"爱"的感情，正式晋升神藏境（大宗师）。

**高潮场景**：
- 与血妖大军的大决战
- 卡洛儿的"世纪婚礼"
- 感悟"爱"晋升大宗师

**本卷成就**：晋升大宗师，击败血妖大军

---

#### 第六卷：回归（650-800章左右）

**核心事件**：黄石公园遗迹 → 御龙班直重建 → 返回地球

**剧情概述**：
黄石公园遗迹开启，这是连接地球和吕宙的重要通道。吕树通过遗迹返回地球，与亲朋好友重逢。

卡洛儿在遗迹中与吕树重逢，记忆部分恢复。

御龙班直正式重建，张卫雨等继续担任核心职位。吕树带着御龙班直返回吕宙。

**关键转折**：
- 与地球势力重新建立联系
- 御龙班直重建完成
- 确认了与卡洛儿的感情

**本卷成就**：完成两界穿梭，确定双线主角身份

---

#### 第七卷：终结（800章至完结）

**核心事件**：大决战 → 斩杀恶念 → 继承神王

**剧情概述**：
最终决战到来。吕树联合各方力量，与血妖大军和吕神恶念展开决战。

云倚、虎执、泪诀三位傀儡师成功刺杀血妖大军指挥官。

张卫雨在决战中力竭而亡，后被石学晋复活。

吕树最终斩杀吕神恶念，为父报仇，继承神王之位。

**结局**：吕树成为吕宙神王，与卡洛儿、吕小鱼等人幸福生活。

## 五、经典场景与爽点分析

### 5.1 核心爽点模式

**毒舌收情绪值**
- 吕树通过毒舌让别人产生负面情绪
- 系统自动收集情绪值
- 情绪值可抽奖、兑换道具
- 典型的"嘴炮型"金手指设定

**扮猪吃虎**
- 吕树经常隐藏实力
- 关键时刻突然爆发
- 敌人措手不及

**反向收徒**
- 李弦一想收吕小鱼为徒被拒
- 吕树反而让李弦一吃了不少瘪

**打脸装逼者**
- 各种自以为是的对手被吕树打脸
- 负面情绪值收获满满

### 5.2 重要爽点场景

| 场景 | 爽点类型 | 效果 |
|------|----------|------|
| 臭豆腐摊位 | 日常积累 | 用毒舌卖臭豆腐赚钱 |
| 道元班冲突 | 打脸爽 | 当众让挑衅者难堪 |
| 长白山之战 | 热血爽 | 与绝顶强者并肩作战 |
| 卡洛儿觉醒 | 爱情爽 | 深情告白与守护 |
| 感悟"爱"晋升 | 境界爽 | 因感情突破瓶颈 |
| 大决战复仇 | 高潮爽 | 为父报仇的终极爽感 |

## 七、爽点模型总结

### 7.1 情绪值系统
```
毒舌攻击 → 他人负面情绪 → 系统收集 → 抽奖/兑换 → 实力提升
```

### 7.2 双世界穿梭
```
地球（日常+感情） ↔ 吕宙（升级+复仇）
```

### 7.3 核心爽感来源
- **嘴炮爽**：毒舌吐槽是核心娱乐来源
- **打脸爽**：各种装逼者被当众打脸
- **升级爽**：系统化的实力提升
- **爱情爽**：与卡洛儿的跨种族爱情
- **复仇爽**：为父报仇的终极满足

## 八、重要副本与遗迹详解

### 8.1 盐湖遗迹

**背景**：海外修行者发现的重大遗迹

**参与者**：地球顶级强者、各国修行势力

**核心事件**：
- 石学晋与安东尼对决
- 石学晋击退对手但油尽灯枯
- 吕树赠予洗髓果实
- 石学晋借此突破成为地球第一个A级（一品）

**意义**：标志着地球修行进入新阶段

### 8.2 长白山之战

**背景**：海外修行者大规模入侵

**参与者**：聂廷、石学晋、吕树、卡洛儿、海外诸势力

**核心事件**：
- 海外修行者发现地球空间脆弱
- 大肆入侵长白山
- 聂廷为保护地球自废根基
- 斩杀信仰理论部主教
- 卡洛儿觉醒奥丁血脉

**意义**：全书最大危机之一，奠定吕树崛起基础

### 8.3 黄石公园遗迹

**背景**：连接地球与吕宙的重要通道

**参与者**：吕树、卡洛儿、御龙班直、各方势力

**核心事件**：
- 遗迹开启，各方进入
- 吕树与卡洛儿重逢
- 御龙班直正式重建
- 吕树带领众人返回吕宙

**意义**：完成两界穿梭，双线叙事收束

### 8.4 吕宙大决战

**背景**：血妖大军全面入侵

**参与者**：吕树、御龙班直、傀儡师、血妖大军

**核心事件**：
- 血妖大军发动总攻
- 云倚、虎执、泪诀刺杀指挥官
- 张卫雨力竭而亡后被复活
- 吕树斩杀吕神恶念

**意义**：全书高潮，复仇主线完成
