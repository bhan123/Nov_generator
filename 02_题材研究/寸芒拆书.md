## 一、作品简介

### 1.1 故事背景

一位身怀飞刀绝技的青年。他因为经脉定型，所以内功上不可能有大的成就。可是后来青年却得到一颗丹药，一颗来自修真界的可以脱胎换骨洗经伐髓的"洗髓丹"……

### 1.2 核心设定

| 设定 | 说明 |
|------|------|
| 飞刀 | 主角核心武器，贯穿全文 |
| 洗髓丹 | 改变命运的丹药 |
| 修真界 | 高等位面 |
| 寸芒 | 飞刀绝技名称 |

### 1.3 故事主线

主角李寻因经脉定型无法修炼内功，获得洗髓丹后脱胎换骨，以飞刀绝技纵横都市与修真界，最终成为巅峰强者。
