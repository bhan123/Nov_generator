## 一、作品简介

### 1.1 故事背景

纪宁死后来到阴曹地府，经判官审前生判来世，投胎到了部族纪氏。这里，有夸父逐日，有后羿射金乌，更有为了逍遥长生，历三灾九劫，纵死无悔的无数修仙者……

纪宁也成为了一名修仙者，开始了他的修仙之路。

### 1.2 核心设定

| 设定 | 说明 |
|------|------|
| 黑白学宫 | 纪宁拜入的学府 |
| 方寸山 | 菩提老祖道场 |
| 芒涯国 | 修行者国度 |
| 终极剑道 | 纪宁最终剑道 |
| 三界 | 人、仙、魔所在 |

### 1.3 故事主线

纪宁从阴曹地府投胎到部族纪氏，以剑道崛起，一路成长为混沌宇宙掌控者，对抗终极至尊伊耶尔，拯救修行者文明。
