## 一、作品简介

### 1.1 故事背景

拥有太古血脉的少年东伯雪鹰，自幼丧父、与弟弟相依为命，为救母亲卷入正魔争斗。其间，雪鹰与神族后裔女子余靖秋相识相恋，两人历经磨难，不畏艰险，成为知心爱人，生死与共。

### 1.2 核心设定

| 设定 | 说明 |
|------|------|
| 太古血脉 | 强大的血脉传承 |
| 虚空行者 | 空间法则修炼体系 |
| 夏族 | 故事主要种族 |
| 源世界 | 修炼的世界 |

### 1.3 故事主线

东伯雪鹰自幼展现超凡实力，为救母亲与魔族战斗，一路成长为领主级别强者，化解源世界灭世危机，复活家人朋友。
